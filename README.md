# Asteroid Blaster

Terminal-based asteroid blaster game for TM4C123GH6PM microcontroller

## Quick Start


* Open in Keil uVision 5 (MDK-ARM) and build
* Flash to TM4C123 board
* Open serial terminal at 9600 baud


## Hardware
- TivaC MCU (TM4c123GH6PM)
- LCD
- Buttons

## Project Stages
- Stage 1: terminal-based game
- Stage 2: hardware-based game
- Stage 3: compile and flash without Keil 
- Stage 4: port to STM32
